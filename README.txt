Movie Trailer Website

This project displays an HTML file with movies, series and documentaries. They are represented with their own posters and once they got clicked,
a youtube trailer link is shown.

Prerequisites:
1- You will need Python 2.7 to run this project. To know if python is installed on your PC just open the terminal and type python. It should show
the python version installed.

If you don't have python installed on your PC, please go to https://docs.python.org/2/install/ to get the installation instructions.

How to run the code:
Once you have python installed, execute the file "Entertainment_center.py". It uses 2 python files, fresh_tomatoes.py which generates an HTML
template and the second file is media.py where all project classes are stored.
